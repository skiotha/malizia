# temp/

Data files used by the bot.

| File | Description |
|------|-------------|
| `abilities.all.json` | Master abilities list (source of truth, not committed) |
| `abilities.lookup.json` | Normalized lookup index consumed by `/lookup` — generated by `normalize-abilities.mjs` |

## Regenerating the lookup index

```bash
node temp/normalize-abilities.mjs
```

This reads `abilities.all.json`, normalizes all tags (lowercase, deduplicated, consolidated), and writes the result to `abilities.lookup.json`.

## abilities.all.json format

Each entry is expected to have at minimum:

```jsonc
{
  "name": "Ability Name",          // required
  "tags": ["tag1", "tag2"],        // required – arbitrary strings
  "description": "..."             // optional – passed through unchanged
  // ...any other fields are passed through unchanged
}
```

## abilities.lookup.json format

```jsonc
[
  {
    "name": "Ability Name",
    "tags": ["normalized-tag", "another-tag"],
    "description": "..."
    // ...any extra fields from the source entry
  }
]
```
